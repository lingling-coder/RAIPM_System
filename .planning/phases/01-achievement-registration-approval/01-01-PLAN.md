    - V1 migration: CREATE TABLE `paper` with all REG-01 fields
      - Fields: id (BIGINT AUTO_INCREMENT), title (VARCHAR 500), authors (VARCHAR 1000), journal (VARCHAR 500), doi (VARCHAR 255), issn (VARCHAR 50), volume (INT nullable), issue (INT nullable), pages (VARCHAR 50), publish_year (YEAR), index_status (VARCHAR 50, values: SCI/SSCI/EI/CPCI/CSCD/CSSCI/北大核心/其他), impact_factor (DECIMAL 10,3 nullable), zone (VARCHAR 20 nullable, values: 一区/二区/三区/四区/无), abstract_text (TEXT nullable)
      - Common audit fields: created_by (BIGINT), created_time (DATETIME), updated_by (BIGINT nullable), updated_time (DATETIME nullable)
      - Version field: version (INT DEFAULT 1) for optimistic locking
      - Status: status (VARCHAR 50, default DRAFT)
      - Department: dept_id (BIGINT, for SQL-layer permission injection)
      - Classified: is_classified (TINYINT DEFAULT 0), classified_level (VARCHAR 20 nullable)
      - Project linkage: project_ref (VARCHAR 500 nullable)
      - Archive number: archive_no (VARCHAR 100 nullable)
      - Indexes: idx_doi (UNIQUE on doi where doi IS NOT NULL), idx_status (status), idx_dept_id (dept_id), idx_created_time (created_time)
    - Paper entity: @TableName("paper"), @Data, fields match migration
    - PaperMapper: extends BaseMapper<Paper>
    - PaperDTO: MapStruct target for create/update input. Include @NotBlank on title/authors/journal/publishYear/indexStatus. Include @Pattern for doi format match `^10\.\d{4,}\/.*$`.

    **PaperService:**
    - createPaper(PaperDTO): validate required fields, set status=DRAFT, set deptId from SecurityUtils, insert
    - updatePaper(Long id, PaperDTO): validate ownership (created_by matches current user OR user is secretary), update fields, set version=version+1 WHERE version=oldVersion
    - submitPaper(Long id): transition status from DRAFT to PENDING_DEPT_REVIEW, check duplicate DOI per D-45 (call checkDuplicateDoi), throw AchievementException if duplicate found
    - saveDraft(PaperDTO): same as create but explicitly set status=DRAFT
    - loadDraft(Long id): load paper where status=DRAFT and created_by=currentUser
    - getPaperById(Long id): load paper, throw if not found AND not authorized (dept_id filtering via Phase 0 interceptor)
    - pagePapers(int page, int size, String status, String keyword): use MyBatis-Plus Page + LambdaQueryWrapper, .eq(status), .like(title|authors, keyword)
    - checkDuplicateDoi(String doi): return true if exists paper with same doi and status != INVALIDATED
    - Use @Transactional on all write methods

    **DOI Integration (achievement-integration):**
    - DoiSourceEnum: CROSSREF("Crossref"), OPENALEX("OpenAlex") with displayName
    - DoiSourcePriorityConfig: @ConfigurationProperties(prefix="doi.source"), List<String> priority (default: crossref,openalex), @Component, provides getOrderedSources() returning List<DoiSourceEnum>
    - DoiLookupResult: boolean found, String doi, String title, String authors, String journal, Integer volume, Integer issue, String pages, Integer publishYear, String abstractText
    - CrossrefClient: RestTemplate-based client. GET https://api.crossref.org/works/{doi}?mailto=admin@institute.cn. Parse JSON response: message.title[0] -> title, message.author[].given+family -> authors (semicolon-separated), message.container-title[0] -> journal, message.volume -> volume, message.issue -> issue, message.page -> pages, message.published-print.date-parts[0][0] -> publishYear, message.abstract -> abstractText (strip HTML). Return Optional<DoiLookupResult>. 5s timeout.
    - OpenAlexClient: RestTemplate-based client. GET https://api.openalex.org/works/doi:{doi}. Parse: display_name -> title, authorships[].author.display_name -> authors (semicolon-separated), primary_location.source.display_name -> journal, volume -> volume, issue -> issue, pages -> pages, publication_year -> publishYear, abstract_inverted_index -> abstractText (de-invert index). 5s timeout.
    - DoiAutoFillService: @Service. lookup(String doi): get ordered sources from priorityConfig, iterate, call lookupFromSource for each. First non-empty result wins (D-14 fallback). Catch exceptions from each source (log warning), continue to next. Return DoiLookupResult.notFound(doi) if all fail.
    - ApiGatewayService: Phase 0-provided service for external API calls (Resilience4j retry wrapper). For Phase 1, create a simplified version that wraps RestTemplate calls with retry (max 3, exponential backoff 1s/2s/4s). Include CircuitBreaker annotation.
    - FallbackHandler: Interface with handle(Throwable t, String source) returning Optional<DoiLookupResult>. Implementation logs error and returns empty.

|----------|-------------|
| browser -> API | Unauthenticated / unvalidated input from browser crosses into backend services |
| backend -> external DOI API | Backend calls Crossref/OpenAlex — potential SSRF if domain not whitelisted |
| file upload -> filesystem | User-uploaded files stored on server filesystem |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-01 | Spoofing | POST /api/papers (unauthenticated) | mitigate | Requires JWT token (Phase 0 Spring Security), @PreAuthorize on controller |
| T-01-02 | Tampering | Paper status field via concurrent update | mitigate | @Version field for optimistic locking, WHERE status=oldStatus in Service |
| T-01-03 | Information Disclosure | File path exposure via download URL | mitigate | Phase 0 proxy service + UUID filenames (P0-06), AttachmentService verifies permission before streaming |
| T-01-04 | Information Disclosure | DOI API response leaking to unauthorized user | mitigate | DOI lookup is authenticated (requires JWT), no access control needed on public DOI metadata |
| T-01-05 | Elevation of Privilege | User modifies another user's paper | mitigate | PaperService.updatePaper verifies created_by == currentUser OR currentUser has secretary role |
| T-01-06 | Denial of Service | Large file upload fills disk | mitigate | 50MB per-file limit + Nginx client_max_body_size 55MB + Spring multipart max-file-size |
| T-01-07 | Spoofing (SSRF) | DOI API call to internal network | mitigate | DOI client whitelist: only crossref.org, openalex.org; URL constructed from enum, not user input |
| T-01-08 | Tampering | Excel formula injection via uploaded file | mitigate | EasyExcel default: formulas not executed. Accept as mitigated. |
