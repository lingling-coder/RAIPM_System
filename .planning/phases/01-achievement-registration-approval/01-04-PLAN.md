    **BatchImportService (per RESEARCH.md Pattern 4):**
    - @Service
    
    - importExcel(MultipartFile file, Long userId):
      - Validate file: must be .xlsx or .xls, max 10MB (reasonable limit for import files)
      - Create BatchImportResult with counters initialized to 0
      - Use EasyExcel.read with AnalysisEventListener<UnifiedImportRow>:
        ```
        EasyExcel.read(file.getInputStream(), UnifiedImportRow.class,
          new AnalysisEventListener<UnifiedImportRow>() {
            private List<UnifiedImportRow> batch = new ArrayList<>();
            private static final int BATCH_SIZE = 500;  // Per Pitfall 3 mitigation
            
            @Override
            public void invoke(UnifiedImportRow row, AnalysisContext context) {
              totalRows++;
              List<String> rowErrors = validateRow(row);
              if (!rowErrors.isEmpty()) {
                errors.add(new ImportError(rowIndex, row.getType(), rowErrors));
                errorRows++;
                return;
              }
              if (isDuplicate(row)) {
                errors.add(new ImportError(rowIndex, row.getType(), List.of("重复数据，已跳过")));
                skippedRows++;
                return;
              }
              batch.add(row);
              if (batch.size() >= BATCH_SIZE) {
                batchInsert(batch);
                batch.clear();
              }
            }
            
            @Override
            public void doAfterAllAnalysed(AnalysisContext context) {
              if (!batch.isEmpty()) {
                batchInsert(batch);
              }
            }
          }).sheet().doRead();
        ```
      - After import: if errors exist, generate error report Excel and store path
      - Save ImportRecord to database
      - Return BatchImportResult

    - validateRow(UnifiedImportRow row) -> List<String>:
      - Common validations: type must match paper/patent/copyright; title not blank
      - Paper: authors not blank, journal not blank, doi format check, publishYear between 1900-current
      - Patent: inventors not blank, applicationNo not blank, patentType in allowed list, country in allowed list, applicationDate parsable
      - Copyright: copyrightHolder not blank, registrationNo not blank, registrationDate parsable, version not blank, softwareCategory in allowed list
      - Return list of error messages (empty = valid)

    - isDuplicate(UnifiedImportRow row) -> boolean:
      - Paper: query paper table by doi (WHERE doi = ? AND status != 'INVALIDATED')
      - Patent: query patent table by application_no
      - Copyright: query copyright table by registration_no
      - Return true if exists

    - batchInsert(List<UnifiedImportRow> batch):
      - For each row, create appropriate entity (Paper/Patent/Copyright) from row data
      - Set status=DRAFT, deptId from current user's department
      - Batch insert using MyBatis-Plus saveBatch
      - Use a fresh SqlSession for each batch (flush/clear between batches for memory)

    - generateErrorReport(Long importRecordId, List<ImportError> errors):
      - Create Excel with columns: 行号 | 类型 | 错误原因
      - Write each ImportError as a row
      - Store at: {importDir}/{importRecordId}_errors.xlsx
      - Update importRecord.errorReportPath

    - downloadTemplate(): delegate to TemplateGeneratorService
